# Introduction to SeaStack

SeaStack helps you manage remote servers and deploy containerized applications using Docker Swarm, all routed behind a Traefik proxy. The goal is a simple, open-source control plane where you can register servers, run commands securely, and orchestrate app deployments on a Swarm cluster with sensible defaults.

<Callout type="note" title="Open Source">
SeaStack is fully open source and contributions are welcome! Check out our [Contribution Guidelines](https://github.com/SeaStackApp/sea-stack-app?tab=readme-ov-file#contribution-guidelines) to get started.
</Callout>

## Key Features

SeaStack provides a comprehensive platform for managing your infrastructure:

- **Server Management**: Register and manage remote Linux servers with secure SSH access
- **Remote Execution**: Execute administrative actions like reboot, terminal access, and system monitoring
- **Container Orchestration**: Deploy and manage applications using Docker Swarm (planned)
- **Traefik Integration**: Zero-downtime routing via automatic Traefik proxy configuration (planned)
- **Modern Web UI**: Built with Next.js, tRPC, Prisma, and PostgreSQL

## Architecture Overview

SeaStack is built as a monorepo using modern web technologies:

- **Frontend**: Next.js with React for a responsive, fast UI
- **API**: tRPC for type-safe API communication
- **Database**: Prisma ORM with PostgreSQL
- **Authentication**: Better Auth for secure user management
- **Build System**: Turborepo for efficient monorepo management

## Use Cases

SeaStack is ideal for:

- **Small Teams**: Manage your infrastructure without complex tooling
- **Side Projects**: Deploy and manage hobby projects with ease
- **Development Environments**: Quickly spin up and manage dev/staging environments
- **Learning**: Understand modern DevOps practices with a real-world application
